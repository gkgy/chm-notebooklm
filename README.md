# chm-notebooklm


我想要把chm的文件弄出来，转换成统一的text包。然后投送到notebooklm

text是最适合大模型来训练学习的，而且不像pdf那么的难

notebooklm最高能一次性放入50个文件。


于是我就做了一个python转换text，设置的50个投喂包



# 使用方式

提前安装python3
有zip解压软件（7zip或者winrar）

## 第一步 首先把chm用7zip解压


## 第二步
把解压的文件夹
用CHM文件夹转文本.py这个软件打开
点击转换就可以使用了
